## Installation

1. Download the [latest release of the translation](https://github.com/DavidK0/AgenciaEspacialdeGatitos/releases/latest) (a single Gauges.xml file)

2. Replace `Kitten Space Agency\Content\Core\Gauges.xml` with the one you downloaded
